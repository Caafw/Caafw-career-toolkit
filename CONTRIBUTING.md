# Contributing to CAAFW Career Toolkit

Thank you for helping improve this skill. Every contribution helps more people plan better careers.

---

## How to Contribute

1. **Fork** the repository
2. **Make your changes** (see types below)
3. **Test** — install your modified `SKILL.md` in a Claude.ai Project and run `/career-toolkit` to verify output quality
4. **Submit a Pull Request** with a clear description of what changed and why

---

## Contribution Types

### Add a New Market
In `SKILL.md`, under each tab specification, add market-specific guidance:
- Job boards specific to that country
- Visa/work authorisation common scenarios
- 10–15 real company names in that market
- Salary benchmark ranges (cite your source)
- Cultural/CV format notes (e.g. photo on CV, different interview norms)

**Priority markets needed:** Singapore, UAE, Germany, Canada, Australia, New Zealand

### Add a New Situation Type
Current: graduate, transition, returner, senior, international, active.

**Ideas for new situations:**
- PhD/academic to industry
- Military/defence to civilian
- Nonprofit to corporate
- Freelance/self-employed returning to employment
- Entrepreneur returning after a startup

### Improve Output Quality
If Claude generates generic or inaccurate output for a specific situation, improve the relevant section of `SKILL.md`. Include a before/after example showing the improvement.

### Translate the Skill
Create `SKILL-[language-code].md` (e.g. `SKILL-hi.md` for Hindi).
Translate the instruction text — keep all code blocks, HTML, and command names in English.

**Priority languages:** Hindi, Filipino/Tagalog, Spanish, Arabic

---

## Quality Standards

- **Real company names only** — no placeholders like "a major bank"
- **Real salary data** — cite your source (LinkedIn Salary, Glassdoor, industry surveys)
- **Tested output** — verify your changes actually produce better output before submitting
- **No hallucinated data** — if uncertain about a fact, leave it out

---

## Code of Conduct

Be kind. Be constructive. We are all here to help people build better careers.
